### Featured project

**[DermRx Agent](https://github.com/meshivanshsinghh/dermrx-agent)** — Built for Google's MedGemma Impact Challenge 2026 (Kaggle). A multi-model medical AI pipeline that orchestrates 3 quantized Google HAI-DEF models (MedSigLIP, MedGemma 4B, TxGemma 2B) on a single T4 GPU using PyTorch and HuggingFace Transformers. Aggregates drug knowledge from 3 federal sources into a unified safety system that catches dangerous drug interactions.
